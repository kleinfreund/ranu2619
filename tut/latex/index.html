Kurzanleitung um LaTeX unter Windows einzurichten.

  1. MiKTeX herunterladen (Net Installer unter Other Downloads wählen.)
  2. MiKTeX installieren
    1. Häkchen bei I accept the MiKTeX copying conditions. → Weiter
    2. Download MiKTeX auswählen → Weiter
    3. Complete MiKTeX auswählen → Weiter
    4. Irgendeinen Mirror in Deutschland auswählen → Weiter
    5. Heißgetränk der Wahl zu bereiten und Mathe machen – das wird dauern.
    6. Wenn der Download fertig ist, den Installer beenden
    7. Installer erneut öffnen und das Prozedere wiederholen
    8. Statt Download wählt ihr jetzt Install MiKTeX
    9. Alles weitere so belassen. Das war’s.
  3. Texmaker herunterladen
  4. Texmaker installieren
    1. I Agree
    2. Install
    3. Fertig – War beinahe schon zu einfach, oder?
