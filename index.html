VPN-Verbindung unter Ubuntu aufbauen

  1. Lade die PCF-Konfiguration für eine externe Verbindung zum Uninetz von dieser Seite herunter. PCF-Dateien speichern Cisco VPN-Einstellungen.
  2. Installiere die Pakete network-manager-vpnc und network-manager-vpnc-gnome: 
    sudo apt-get update
sudo apt-get install network-manager-vpnc network-manager-vpnc-gnome
  3. Eine bereits konfigurierte VPN-Verbindung kann nun über den Network Manager direkt importiert werden. Klicke oben rechts im Panel (also in der Leiste oben) auf das Symbol für Netzwerkverbindungen und dann auf Verbindungen bearbeiten....
  4. Klicke auf Hinzufügen und wähle die Option für den Import einer VPN-Konfiguration
  5. Wähle in den folgenden Schritten die zuvor heruntergeladene PCF-Datei aus und konfiguriere noch den Uni-Benutzernamen. Das VPN kann nun gespeichert und verwendet werden.
