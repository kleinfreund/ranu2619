Aufgabe 1

Berechnen Sie die Wahrscheinlichkeiten aller möglichen Augensummen beim gleichzeitigen Würfeln mit drei Würfeln.

Aufgabe 2

Wie groß ist die Wahrscheinlichkeit p, bei fünf Würfen (k = 5) mit einem idealen Würfel (n = 6) fünf verschiedene Ergebnisse zu haben?

Lösung:

Beim ersten Wurf hat man in jedem Fall eine Zahl gewürfelt, die vorher noch nicht aufgetreten ist. Für diesen Wurf ist p1 = \( \frac{ 6 }{ 6 } \). Für jeden weiteren Wurf verringert sich die Zahl entsprechend: p2 = \( \frac{ 5 }{ 6 } \), p3 = \( \frac{ 4 }{ 6 } \), usw.

\[ p = \frac{ 6 }{ 6 } \cdot \frac{ 5 }{ 6 } \cdot \frac{ 4 }{ 6 } \cdot \frac{ 3 }{ 6 } \cdot \frac{ 2 }{ 6 } = \frac{ 720 }{ 7776 } = 0.0926 = 9.26 \% \] Eine allgemeine Formel für diesen Vorgang ist diese: \[ \frac{ n! }{ n^{k} } \]

Aufgabe 3

Überprüfen Sie Ihre Kenntnisse über die Binomialkoeffizienten und das Pascalsche Dreieck. Beweisen Sie die für natürliche Zahlen n, k (1 ≤ k ≤ n) gültige Beziehung.

\begin{align*} \binom{ n }{ k-1 } + \binom{ n }{ k } = \binom{ n+1 }{ k } \end{align*}

Was hat diese mit dem Pascalschen Dreieck zu tun? Wiederholen Sie den Binomischen Lehrsatz. Beweisen Sie unter dessen Verwendung, dass jede n-elementige Menge genau \( 2^{n} \) Teilmengen besitzt. (Anmerkung: Auf Grund dieses Resultats bezeichnet man die Menge aller Teilmengen als Potenzmenge.) In welchem Sinne kann man dieses Resultat als Variation mit Wiederholung auffassen?

Lösung:

n sind dabei die Zeilen und k die Spalten des Dreiecks. In Zeile 1 gibt es nur eine Spalte, in Zeile 2 gibt es 2 Spalten, usw.

n = 1 │                            1
    2 │                         1     1
    3 │                      1     2     1
    4 │                   1     3     3     1
    5 │                1     4     6     4     1
    6 │             1     5    10    10     5     1
    7 │          1     6    15    20    15     6     1
    8 │       1     7    21    35    35    21     7     1
    9 │    1     8    28    56    70    56    28     8     1

Setzt man nun die Werte für Zeilen und Spalten entsprechend in die Formel für die Binomialkoeffizienten ein, erhält man die Werte aus dem Pascalschen Dreieck.

\[ \binom{ 0 }{ 0 } = 1 \\ \binom{ 1 }{ 0 } = 1, \binom{ 1 }{ 1 } = 1 \\ \binom{ 2 }{ 0 } = 1, \binom{ 2 }{ 1 } = 2, \binom{ 2 }{ 2 } = 1 \\ \binom{ 3 }{ 0 } = 1, \binom{ 3 }{ 1 } = 3, \binom{ 3 }{ 2 } = 3, \binom{ 3 }{ 3 } = 1 \\ \binom{ 4 }{ 0 } = 1, \binom{ 4 }{ 1 } = 4, \binom{ 4 }{ 2 } = 6, \binom{ 4 }{ 3 } = 4, \binom{ 4 }{ 4 } = 1 \\ \]

Mit dem Dreieck lassen sich einfach höhergradige Potenzen über Klammern auflösen. Die Faktoren kann man dabei direkt für den Grad des Ausdrucks (zum Beispiel n = 4) ablesen:

\[ \begin{align*} (a + b)^{4} &= \binom{ 4 }{ 0 } a^{4}b^{0} + \binom{ 4 }{ 1 } a^{3}b^{1} + \binom{ 4 }{ 2 } a^{2}b^{2} + \binom{ 4 }{ 3 } a^{1}b^{3} + \binom{ 4 }{ 4 } a^{0}b^{4} \\ \\ &= a^{4} + 4a^{3}b + 6a^{2}b^{2} + 4ab^{3} + b^{4} \\ \end{align*} \]

Aufgabe 4

Eine Schachtel enthalte sechs Eier, von denen zwei schlecht seien (was äußerlich nicht erkennbar ist). Für die Zubereitung eines Eierkuchens werden drei Eier entnommen und in die Pfanne geschlagen. Mit welcher Wahrscheinlichkeit ist der Eierkuchen essbar?

Lösung:

Aus sechs Eiern wählt man 2 aus (\( \binom{ 6 }{ 2 } \)) – das ist die Gesamtanzahl der Möglichkeiten. Der „günstige“ Fall ist nun aus den vier guten Eiern drei auszuwählen (\( \binom{ 4 }{ 3 } \))

\[ \frac{ \binom{ 4 }{ 3 } }{ \binom{ 6 }{ 3 } } = \frac{ 4 }{ 20 } = 0.2 = 20 \% \]

Alternative Lösung:

Ai: „i-tes Ei ist einwandfrei“ (i = 1, 2, 3)

\[ P(A_{1} \cap A_{2} \cap A_{3}) = \frac{ 4 }{ 6 } \cdot \frac{ 3 }{ 5 } \cdot \frac{ 2 }{ 4 } = \frac{ 1 }{ 5 } = 0.2 = 20 \% \]

Aufgabe 5

Eine Trainingsgruppe besteht aus neun Jungen und drei Mädchen. Für eine Delegation werden zufällig vier Personen ausgewählt. Wie groß ist die Wahrscheinlichkeit, dass

  • Mindestens ein
  • Genau ein

Mädchen ausgewählt wurde?

Lösung:

Für den ersten Teil der Aufgabe nehmen wir das Gegenereignis zur Hilfe, nämlich dass kein Mädchen gewählt wird und ziehen es von 1 ab, um auf das gesuchte Ergebnis zu kommen.

\[ 1 - \overline{p} = 1 - \frac{ \binom{ 9 }{ 4 } \cdot \binom{ 3 }{ 0 } }{ \binom{ 12 }{ 4 } } = 1 - \frac{ 126 }{ 495 } = 1 - 0.25\overline{45} = 0.74\overline{54} = 74.55 \% \]

Für den zweiten Teil ist es einfacher, sich das gewünschte Ereignis direkt anzuschauen.

\[ p = \frac{ \binom{ 9 }{ 3 } \cdot \binom{ 3 }{ 1 } }{ \binom{ 12 }{ 4 } } = \frac{ 84 \cdot 3 }{ 495 } = 0.5\overline{09} = 50.91 \% \]
