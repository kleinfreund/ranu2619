7.2 Bitverarbeitung

Nachdem die erste Variante zwar mit bitweisen Operatoren arbeitete, aber das Gesamtresultat dennoch mit regulärer Addition zusammenzählte und auch eine Zählvariable inkrementiert wurde, ist die zweite Variante „etwas“ einfacher gestrickt.

Was ich nicht wusste, war, dass die bitweisen Operatoren auch auf normalen Integer-Werten arbeiten und man mit XOR, AND und Links-Shift (<<) bereits die zwei Bitstrings erhält, die man braucht, um das Ergebnis zu ermitteln.

Java-Implementierung binärer Addition:

public static int addBinary(int a, int b) {
int carry = (a & b) << 1;
int result = a ^ b;
return ((result & carry) != 0) ? addBinary(result, carry) : (result ^ carry);
}

Vollständiger Quellcode: BinaryAddition.java.
