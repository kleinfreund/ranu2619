4.1 LED-Würfel

Wie im Schaltbild zu sehen ist, fließt durch die Spannungsquelle ein Strom Iges in den Knoten F. Dieser fließt aus dem Knoten A wieder heraus. Alle Widerstände im Würfel sind gleich groß und ihre Anordnung ist außerdem symmetrisch. Daraus ergibt sich, dass in den Kanten, die von F ausgehen und in den Kanten, die in A zusammenkommen, jeweils ein Drittel des Gesamtstromes fließen.

Iges = I6 + I7 + I10
⇒ I6 = I7 = I10 = \( \frac{ 1 }{ 3 } \) Iges

Iges = I1 + I4 + I11
⇒ I1 = I4 = I11 = \( \frac{ 1 }{ 3 } \) Iges

⇒ I1 = I4 = I6 = I7 = I10 = I11 = \( \frac{ 1 }{ 3 } \) Iges

Aufgrund der weiteren Verzweigung der Kanten im Würfel kann man diese relativen Aussagen auch für die weiteren Ströme angeben:

I1 = I2 + I5
I4 = I3 + I12
I6 = I5 + I8
I7 = I2 + I5
I10 = I9 + I12
I11 = I8 + I9

⇒ I2 = I3 = I5 = I8 = I9 = I12 = \( \frac{ 1 }{ 6 } \) Iges

a) Widerstand RAF in der Raumdiagonale

Um zu berechnen, wie groß der Widerstand RAF ist, benötigen wir die Spannung, die durch den Würfel fließt. Aufgrund der Eigenschaften des Würfels und der gleichgroßen Widerstände, ist es für die Spannung egal, auf welchem Weg der Strom durch den Würfel geht.

\[\begin{align*} U_{AF} &= R_{AF} \cdot I_{ges} \\ \Rightarrow R_{AF} &= \frac{ U_{AF} }{ I_{ges} } \end{align*}\]

Weiterhin gilt Ri = R1 = … = R12.

\[\begin{align*} U_{AF} &= R_{1}I_{1} + R_{2}I_{2} + R_{7}I_{7} \\ U_{AF} &= RI_{1} + RI_{2} + RI_{7} \\ U_{AF} &= R \cdot (I_{1} + I_{2} + I_{7}) \\ U_{AF} &= R \cdot \left( \frac{ 1 }{ 3 } I_{ges} + \frac{ 1 }{ 6 } I_{ges} + \frac{ 1 }{ 3 } I_{ges} \right) \\ U_{AF} &= R \cdot \frac{ 5 }{ 6 } I_{ges} \\ \end{align*}\] \[\begin{align*} \Rightarrow R_{AF} &= \frac{ R \cdot \frac{ 5 }{ 6 } I_{ges} }{ I_{ges} } \\ R_{AF} &= R \cdot \frac{ 5 }{ 6 } \\ R_{AF} &= \frac{ 5 }{ 6 } \text{ kΩ} \\ \end{align*}\]

b) Widerstand RAC in der Flächendiagonale

\begin{align*} R_{3,6} &= R_{3} + R_{6} = \frac{ R }{ 2 } + \frac{ R }{ 2 } = R \\ \\ R_{1,2,5,7} &= R_{1} + R_{2} + R_{5} + R_{7} = R + \frac{ R }{ 2 } + \frac{ R }{ 2 } + R = 3R \\ \\ R_{ges} &= R_{3,6}||R_{1,2,5,7} = \frac{ R \cdot 3R }{ R + 3R } = \frac{ R \cdot 3R }{ 4R } = \frac{ 3R }{ 4 } = \frac{ 3 }{ 4 } \text{ kΩ} \\ \end{align*}

c) Widerstand RAB in der Kante

Da es in dem Würfel Punkte gleichen Potentials gibt, besteht die Möglichkeit ein paar Knoten zusammenzuschalten. So erhält man ein alternatives Schaltbild, indem man die Punkte D, H und C, E zu jeweils einem Knoten (DH und CE) zusammenschaltet.

Nun gibt sind die meisten Widerstände parallel geschaltet. Ihren Widerstand kann man demnach gebündelt als die Hälfte von R angeben.

\begin{align*} R_{1,2,3} &= R_{1} + R_{2} + R_{3} = \frac{ R }{ 2 } + R + \frac{ R }{ 2 } = 2R \\ \\ R_{1,2,3,4} &= R_{1,2,3}||R_{4} = \frac{ 2R \cdot \frac{ R }{ 2 } }{ 2R + \frac{ R }{ 2 } } = \frac{ R }{ \frac{ 5 }{ 2 }R } = \frac{ 2 }{ 5 }R \\ \\ R_{1,2,3,4,5,6} &= R_{1,2,3,4} + R_{5,6} = \frac{ 2 }{ 5 }R + 2 \cdot \frac{ R }{ 2 } = \frac{ 7 }{ 5 }R \\ \\ R_{ges} &= R_{1,2,3,4,5,6,7} = R_{1,2,3,4,5,6}||R_{7} = \frac{ \frac{ 7 }{ 5 }R \cdot R }{ \frac{ 7 }{ 5 }R + R } = \frac{ \frac{ 7 }{ 5 }R \cdot R }{ \frac{ 12 }{ 5 }R } = \frac{ \frac{ 7 }{ 5 } }{ \frac{ 12 }{ 5 } } R = \frac{ 7 }{ 12 }R \\ &= \frac{ 7 }{ 12 } \text{ kΩ} \\ \end{align*}

4.5 Pegel-Spannungsteiler (dB)

a) + b) Reduzierte Spannungen Un und erforderliche Widerstände Rn

dBu ist der Spannungspegel, der sich mit folgender Formel berechnen lässt:

\[\begin{align*} L_{U} &= 20 \cdot \log \left( \frac{ U }{ U_{0} } \right) \\ \Rightarrow \frac{ L_{U} }{ 20 } &= \log \left( \frac{ U }{ U_{0} } \right) \\ \Rightarrow 10^{ \frac{ L_{U} }{ 20 } } &= \frac{ U }{ U_{0} } \\ \Rightarrow U_{0} \cdot 10^{ \frac{ L_{U} }{ 20 } } &= U \\ \end{align*}\]

Die Bezugsgröße U0 für den Spannungspegel ist festgelegt auf \( 0.7746 \text{ V} \). Das ergibt sich aus folgender Rechnung.

Bezugsgröße:

\begin{align*} U_{0} &= \sqrt{ 600 ~Ω \cdot 1 \text{ mW} } \\ &= \sqrt{ 600 ~Ω \cdot 0.001 \text{ W} } \left[ \sqrt{Ω \cdot mW} = \sqrt{\frac{ V }{ A } \cdot V \cdot A} = \sqrt{V^{2}} = V \right] \\ &= \sqrt{ 0.6 } \text{ V} \\ &= 0.7746 \text{ V} \\ \end{align*}

Weiterhin gilt für U und Un folgendes:

\[\begin{align*} U &= 0.7746 \text{ V} \cdot 10^{ \frac{ L_{U} }{ 20 } } \\ \end{align*}\] \begin{align*} U_{n} &= \frac{ R_{n}||R_{L} }{ R_{n}||R_{L} + R_{i} } \cdot U_{0} \\ \\ \frac{ U_{n} }{ U_{0} } &= \frac{ \frac{ R_{n}R_{L} }{ R_{n}+R_{L} } }{ \frac{ R_{n}R_{L} }{ R_{n}+R_{L} } + R_{i} } \\ \\ \frac{ U_{n} }{ U_{0} } &= \frac{ \frac{ R_{n}R_{L} }{ R_{n}+R_{L} } }{ \frac{ R_{n}R_{L} + R_{i}(R_{n}+R_{L}) }{ R_{n}+R_{L} } } \\ \\ \frac{ U_{n} }{ U_{0} } &= \frac{ (R_{n}R_{L}) \cdot (R_{n}+R_{L}) }{ (R_{n}+R_{L}) \cdot (R_{n}R_{L} + R_{i}(R_{n}+R_{L}))} \\ \\ \frac{ U_{n} }{ U_{0} } &= \frac{ 1 }{ 1 + \frac{ R_{i}(R_{n}+R_{L}) }{ R_{n}R_{L} } } \\ \\ \Rightarrow \frac{ 1 }{ \frac{ U_{n} }{ U_{0} } } &= 1 + \frac{ R_{i}(R_{n}+R_{L}) }{ R_{n}R_{L} } \\ \\ \frac{ U_{0} }{ U_{n} } &= 1 + \frac{ R_{i}(R_{n}+R_{L}) }{ R_{n}R_{L} } \\ \\ \frac{ U_{0} }{ U_{n} } &= \frac{ R_{n}R_{L} + R_{i}(R_{n}+R_{L}) }{ R_{n}R_{L} } \\ \\ \frac{ U_{0} }{ U_{n} } &= \frac{ R_{n}R_{L} }{ R_{n}R_{L} } + \frac{ R_{i}R_{n} }{ R_{n}R_{L} } + \frac{ R_{i}R_{L} }{ R_{n}R_{L} } \\ \\ \frac{ U_{0} }{ U_{n} } &= 1 + \frac{ R_{i} }{ R_{L} } + \frac{ R_{i} }{ R_{n} } \\ \\ \frac{ R_{i} }{ R_{n} } &= \frac{ U_{0} }{ U_{n} } - \frac{ R_{i} }{ R_{L} } - 1 \\ \\ \frac{ R_{i} }{ R_{n} } &= \frac{ (U_{0}R_{L}) - (U_{n}R_{i}) - (U_{n}R_{L}) }{ U_{n}R_{L} } \\ \\ \frac{ R_{i} }{ R_{n} } &= \frac{ (-1) \cdot \left( R_{L}(U_{n} - U_{0}) + (U_{n}R_{i}) \right) }{ U_{n}R_{L} } \\ \\ \Rightarrow R_{n} &= \frac{ R_{i} }{ \frac{ (-1) \cdot \left( R_{L}(U_{n} - U_{0}) + (U_{n}R_{i}) \right) }{ U_{n}R_{L} } } \\ \\ R_{n} &= \frac{ R_{i}U_{n}R_{L} }{ (-1) \cdot \left( R_{L}(U_{n} - U_{0}) + (U_{n}R_{i}) \right) } \\ \\ R_{n} &= \frac{ -U_{n} \cdot R_{L} \cdot R_{i} }{ R_{L}(U_{n} - U_{0}) + (U_{n} \cdot R_{i}) } \\ \\ \end{align*}
